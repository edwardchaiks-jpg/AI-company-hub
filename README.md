# AI-company-hub
## 🎯 项目总目标
打造**全链路AI自动化运营平台**，统一管理所有项目、代码、发布，服务两大核心业务：
1. **多行业内容生产+全平台自动发布**：适配马来西亚本地餐饮、房产、零售、服务等行业，覆盖所有主流社媒+视频平台
2. **商家办公流程自动化**：单据拍照/OCR识别→自动录入系统→对账出报表，减少重复人工、降低出错率

## 🧩 10个AI员工最终分工
### 📚 内容生产组
1. **行业研究员**：抓取本地新闻、政策、行业数据，同步筛选YouTube热门方向、高流量话题
2. **文案策划师**：产出全平台适配内容：社媒短文案、小红书种草文、YouTube完整脚本/标题/标签、通知话术
3. **视觉+视频设计师**：制作全尺寸配图、YouTube缩略图/封面、基础视频素材建议
4. **内容审核员**：统一检查所有内容合规性、本地语气适配、信息准确性，不合格打回重改

### 📤 全平台发布组
5. **Meta生态专员**：Facebook/Instagram/Threads 发帖、回复互动、数据统计
6. **综合分发专员**：X(Twitter)/小红书/YouTube 发布、SEO优化、评论管理、热门联动
7. **即时通讯助理**：WhatsApp/Telegram 接收指令、推送结果、审批确认、客户通知

### ⚙️ 业务与总控组
8. **单据+客户管理专员**：维护客户资料库（档案/状态/历史）；接收订单、核对需求；自动生成发票/收款单/对账清单；识别上传单据、自动录入会计系统、标记付款状态
9. **流程调度+订单跟进员**：分配任务、跨岗交接、异常处理；跟进每个订单从接单→付款→服务→对账的完整流转
10. **CEO总助**：对接你的指令、审批关键内容；汇总全平台数据、所有订单进度、客户情况、账目报表；异常第一时间通知你

## 🔗 技术与隐私规则
- 前端：React + Vite
- AI接入：OpenRouter 统一调用GPT/Gemini/Mistral等
- 自动化：n8n / bots.net
- 托管：Vercel 自动部署
- ⚠️ 重要：API密钥、Token、密码等敏感信息**仅存平台环境变量，绝不存入本仓库**
